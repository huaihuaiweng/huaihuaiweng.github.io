Database Implementation Projects

A showcase of my database projects.

Buffer Manager

Implement a buffer manager that uses the 2Q replacement strategy. The implementation takes 64 bit page ids and return pages that can be locked (shared or exclusive), read and written. It supports different page and buffer sizes and is able to fix and unfix pages concurrently from different threads

B+-Tree

Implement the B+Tree data structure that supports lookup, insert, and erase operations.

Operators

Implement buzzdb physical operators, including `Print()`, `Projection()`, `Select()`, `Sort()`, `HashJoin()`, and `HashAggregation()`
